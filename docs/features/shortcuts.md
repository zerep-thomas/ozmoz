# Shortcuts & Controls

Ozmoz utilizes global system hooks (`pynput`) to ensure hotkeys function regardless of which application is currently in focus.

## Global Hotkeys

| Action               | Default Shortcut | Function                                                                                        |
| :------------------- | :--------------- | :---------------------------------------------------------------------------------------------- |
| **Dictation Toggle** | `Ctrl` + `X`     | Starts/Stops voice recording. The transcription is automatically pasted at the cursor position. |
| **AI Analysis**      | `Ctrl` + `Q`     | Sends the currently selected text (or current voice input) to the active LLM for processing.    |
| **Screen Vision**    | `Alt` + `X`      | Captures the active window and opens the Vision prompt for analysis.                            |
| **Web Search**       | `Alt` + `W`      | Initiates a web search query based on voice input or selection.                                 |
| **Toggle UI**        | `Ctrl` + `Alt`   | Shows or hides the floating overlay widget.                                                     |

## Navigation

- **Settings:** Accessible via the gear icon ⚙️ in the main interface.
- **History:** Scroll up in the chat window to view previous interactions.
